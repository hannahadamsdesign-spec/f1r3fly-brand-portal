# F1R3FLY.IO — LinkedIn Post Templates (1080×1080)

Ready-to-post square images for the F1R3FLY.io LinkedIn page. **The PNG files in this
folder are the ones to post** — download, attach to a LinkedIn post, done. No fonts or
design tools needed; all brand typography is built into the files.

## Which template for which post

| Use this | When posting |
|----------|--------------|
| `announcement-*` | Product updates, launches, releases |
| `article-*` | New blog post or long-form content (headline + byline) |
| `quote-*` | A quote from Greg or leadership (with attribution) |
| `event-*` | Conference talks, speaker announcements |
| `stat-*` | A big number (throughput, performance) |
| `tagline-*` | Brand statements ("Concurrent by design, secure by math.") |

Each layout comes in 2–3 background variants (`dark-orb`, `section-teal`,
`section-purple`, `brand-ellipse`, `light`) — pick by tone, mix across the feed.

## Changing the text

The text in these files is placeholder copy from the brand system. **To get a version
with your own headline/copy, ask Hannah** — turnaround is quick, and every variant stays
on-brand (typography, layout, and colors are locked to the F1R3FLY design system).

`vector-source/` holds the SVG and PDF versions (for designers/print). Display
headlines use Brandon Grotesque/Text converted to outlines, so they render identically
everywhere without font licenses.
