# F1R3FLY Brand Fonts

The free fonts in the F1R3FLY type system. Download and install (Mac: double-click →
"Install Font"; Windows: right-click → "Install") so F1R3FLY materials render correctly.

## The system

| Font | Role | Where to get it |
|------|------|-----------------|
| **Brandon Grotesque / Brandon Text** | Display headlines and brand lockups — use when possible | Commercial (HVD Fonts) — **not in this repo.** Activate via [Adobe Fonts](https://fonts.adobe.com/fonts/brandon-grotesque) (included with any Creative Cloud plan), or ask Hannah |
| **Josefin Sans** (5 weights, here) | Default substitute for Brandon; eyebrows, subheaders, UI text | This folder, or [Google Fonts](https://fonts.google.com/specimen/Josefin+Sans) — free |
| **Source Sans 3** (3 weights, here) | Body copy, supporting text | This folder, or [Google Fonts](https://fonts.google.com/specimen/Source+Sans+3) — free |

**Rule of thumb:** Brandon when possible, Josefin when not. If you have Creative Cloud,
activate Brandon Grotesque from Adobe Fonts and use it for display type. If you can't
load Brandon, Josefin Sans is the approved fallback — never substitute anything else.
Body text is always Source Sans 3.

The distributed templates in `/templates/` have Brandon display text converted to
outlines, so they render correctly even without any fonts installed.
