# F1R3FLY Brand Fonts

The complete F1R3FLY type system. Download and install (Mac: double-click → "Install Font";
Windows: right-click → "Install") so F1R3FLY materials render correctly on your machine.

## The system

| Font | Role | License |
|------|------|---------|
| **Brandon Grotesque** (Regular, Bold) | Display headlines and brand lockups — use when possible | HVD Fonts, licensed — **internal F1R3FLY use only** |
| **Brandon Text** (Thin → Black, + italics) | Display at lighter weights, refined text settings | HVD Fonts, licensed — **internal F1R3FLY use only** |
| **Josefin Sans** (Thin → Bold) | Default substitute for Brandon; eyebrows, subheaders, UI text | Free (SIL Open Font License, Google Fonts) |
| **Source Sans 3** (Light, Regular, Bold) | Body copy, supporting text | Free (SIL Open Font License, Google Fonts) |

**Rule of thumb:** Brandon when possible, Josefin when not. If you're making anything
with display type and have the Brandon files installed, use Brandon. If a tool or
environment can't load Brandon, Josefin Sans is the approved fallback — never substitute
anything else. Body text is always Source Sans 3.

The distributed templates in `/templates/` have Brandon display text converted to
outlines, so they render correctly even without any fonts installed.
